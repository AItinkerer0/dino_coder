# dino_coder

🦕 **공룡 디펜스 로그라이트** — 떨어지는 버그를 막으며 코드(LOC)를 길게 작성하는 픽셀 아트 게임. [claude_coder](https://github.com/xixwxx/claude_coder)의 공룡 에디션이다.

바닥의 공룡이 좌우로 움직이며 위에서 떨어지는 버그를 자동으로 막는다. 죽으면 ⭐로 영구 강화 → 다시 도전. **5000 LOC = 배포 성공(승리)**.

- 주인공이 공룡, 배경도 온통 공룡으로 도배
- 무기 17종(한 판에 1개 장착) · 영구 스탯 업그레이드 · 테마 5종
- ←→ 또는 A/D 이동, 공격은 자동. **폰 터치 플레이 지원**
- 바닐라 JS + Canvas 단일 `index.html` (픽셀 폰트만 CDN)

플레이: https://xixwxx.github.io/dino_coder/
